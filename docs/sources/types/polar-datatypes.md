# Polar datatypes

When asked to provide polar datatypes as strings anywhere in Beetl these are the valid values.

- Decimal
- Float32
- Float64
- Int8
- Int16
- Int32
- Int64
- UInt8
- UInt16
- UInt32
- UInt64
- Date
- Datetime
- Duration
- Time
- List
- Array
- Categorical
- Enum
- Struct
- Object
- Utf8
- String
